## 技术决策记录

### 决策 1: 使用 Claude API（云端）
**理由**: 本地模型开发成本高（4-6 周），Claude API 性能稳定
**影响**: 砍掉 ONNX 量化方案，节省 3 周开发时间

### 决策 2: 向量检索仅用于 Chat/搜索
**理由**: 代码补全场景不适合向量检索，应使用 Tree-sitter 符号表
**影响**: 简化向量检索实现，降低复杂度

### 决策 3: 使用 p-queue 替代自研调度器
**理由**: 自研调度器开发成本高，p-queue 成熟稳定
**影响**: 节省 3-4 周开发时间

### 决策 4: 添加轻量协作功能
**理由**: 纯性能优化缺少差异化，轻量协作开发成本可控（2 周）
**影响**: 总周期延长至 20 周，产品定位更清晰
