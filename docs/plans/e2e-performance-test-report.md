- 使用真实文件系统和 chokidar 监听
- 临时目录隔离（`test-temp-{timestamp}`）
- 异步等待确保事件完成
- 清理延迟避免竞态条件

- CodeParseService 继承 EventEmitter 支持事件监听
- 添加 'parsed' 事件用于测试验证
- afterEach 中添加 100ms 延迟确保异步操作完成
